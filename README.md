🎯 What it is
Personal Portfolio Website for showcasing your skills, projects, and experience
Interactive 3D Elements using Three.js (rotating Earth, computer models, stars)
Responsive Design that works on desktop, tablet, and mobile
Live Contact Form integrated with EmailJS
🛠️ Tech Stack
Frontend: React 18 + Vite (fast development and builds)
3D Graphics: Three.js, React Three Fiber, React Three Drei
Styling: Tailwind CSS (utility-first CSS framework)
Animations: Framer Motion (smooth transitions)
Icons: React Icons
Deployment: GitHub Pages
Main Sections
Hero: Landing page with 3D background and typewriter effect
About: Personal introduction
Experience: Timeline of work history
Tech: Skills and technologies
Works: Project showcase with images and links
Contact: Form to send messages
